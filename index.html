Projects

Bump

A “smart-braille” IoT project empowers visually impaired people to become independent. Bump consits of a 2 X 3 matrix of solenoids to display braille characters & receives data through SMS, Google vision, or web scraping.

RehabiliNation

A device that provides a way to make the often long and difficult process of physical therapy fun and enjoyable. Used an arduino with gyroscopic sensors and ported it to keyboard input.

Breadcrumbz

A web app that incorporates volunteering into your daily commute. Created a web service using Node.js and Express. Google Maps API was used for plotting routes and Twilio’s API for delivery verification.

Particle Collision

An event driven particle collision simulation in an enclosed 2-dimensional space enabled by a priority queue using heapsort. This application is written in C#.

Casinorama

A virtual casino containing a host of classic games such as poker, roulette, and blackjack, each with their own AI to play against. This was developed in Java.

Skinny Road

A simple game where you are a cube dodging obstacles of varying sizes coming towards you on a skinny road. This personal project was developed using Unity.
